Ensayos

Bos realizó un ensayo, bajo el título Redirecting URLs with fragment IDs, donde propone una solución para permitir HTTP redireccione si el fragmento de las IDs cambia.

Sostiene que las costantes sibólicas de CSS no debería ser usados y lo defiende en Why “variables” in CSS are harmful

Escribió una reflexión sobre los principios de las especificaciones de buen diseño: “What is a good standard?”

Finalmente, posee una propuesta para una posible tecnología que funcione mejor que los Java applets: Web applications.
